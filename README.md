# Portail virtuel architectural 
Dans le cadre du cours 8INF309, ce projet vise à concevoir un portail virtuel architectural reliant le site de l'UQAC à Chicoutimi et l'école du NAD à Montréal.  
L'objectif est de créer une installation permettant une interaction en temps réel entre les deux lieux, combinant vidéo, son, intelligence artificielle et effets physiques.  
Le projet est réalisé de manière autonome en collaboration avec l'école du NAD sur une période de 14 semaines et est suivi par l'enseignant Damien Brun.
