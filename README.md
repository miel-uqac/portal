# Portail virtuel architectural 

## Description du projet

Dans le cadre du cours [8INF309](https://programmes.uqac.ca/8INF309), ce projet vise à concevoir un portail virtuel architectural reliant le site de l'[UQAC à Chicoutimi](https://www.uqac.ca/) et l'école du NAD à Montréal.  
L'objectif est de créer une installation permettant une interaction en temps réel entre les deux lieux, combinant vidéo, son, intelligence artificielle et effets physiques.  
Le projet est réalisé de manière autonome en collaboration avec l'école du NAD sur une période de 14 semaines et est suivi par l'enseignant Damien Brun.

## Installation

### Pré-requis

- [TouchDesigner](https://derivative.ca/) (version 2025)
- [Tailscale](https://tailscale.com/)
- [NDI Tools](https://ndi.video/tools/)
- [Ollama](https://ollama.com/) (+ espace mémoire pour les modèles si utilisation locale)

### Démarrage

Tailscale doit être lancé sur tous les appareils, NDI doit être configuré et Ollama doit aussi être lancé pour pouvoir lancer le fichier `.toe`.
